# 🏥 Healthcare Management System

A menu-driven Hospital Management System developed using **Python** and **MySQL** as part of a Scrum-based team project.

## 📌 Features

### 👤 Patient Management
- Add Patient
- View All Patients
- Search Patient
- Update Patient
- Delete Patient

### 👨‍⚕️ Doctor Management
- Add Doctor
- View All Doctors
- Search Doctor

### 📅 Appointment Management
- Book Appointment
- Cancel Appointment
- Complete Appointment
- View Appointments

### 💳 Billing Management
- Generate Bill
- View Bills
- Search Bills
- Update Payment Status

### 📊 Reports
- Patient Reports
- Doctor Reports
- Appointment Reports
- Billing Reports

## ▶️ Installation

### Clone the repository

```bash
git clone https://github.com/Harshal-789/Healthcare-Management-System.git
```

### Navigate to the project

```bash
cd Healthcare-Management-System
```

### Install dependency

```bash
pip install mysql-connector-python
```

### Configure database

Update the MySQL credentials in:

```
database_connection.py
```

### Run the project

```bash
python main.py
```
